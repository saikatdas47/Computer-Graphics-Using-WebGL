# Computer-Graphics-Using-WebGL 🎨🖼️🗿

This project demonstrates a **3D virtual museum scene** built using **Three.js (WebGL)**.  
It features an **animated painting with custom shaders**, a **realistic statue model**,  
**dynamic lighting with moving spotlights and shadows**, and full **interactive controls**.

## ✨ Features  

- **Custom Shaders** 🎨  
  - Painting with two modes: “Disco blinking” & “Flowing rainbow stripes”.  
  - Shader-based checkerboard floor with toggleable color schemes.  

- **3D Statue** 🗿  
  - Loaded from **glTF model**.  
  - Physically based materials (metalness & roughness).  
  - Casts and receives realistic shadows.  

- **Dynamic Lighting** 💡  
  - Moving **spotlight** that orbits around the statue.  
  - Ambient + Hemisphere lights for balanced illumination.  

- **Camera & Projection** 🎥  
  - Perspective projection.  
  - OrbitControls for zoom, pan, and rotate.  
  - Autorotation toggle (`R` key).  

- **Textures & Environment** 🖼️  
  - Textured walls and golden frame for painting.  
  - Shadow-catcher floor for realistic lighting.  

- **Interactivity** ⌨️🖱️  
  - Keyboard & mouse controls:  
    - `R` → Toggle autorotate.  
    - `Q` → Switch floor to green/yellow.  
    - `W` → Switch floor to black/white.  
    - `T` → Toggle painting animation style.
